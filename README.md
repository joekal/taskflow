# TaskFlow

TaskFlow aide les freelances et petites équipes à organiser leurs projets, collaborer en temps réel et optimiser leurs journées grâce à des algorithmes intelligents.

## Description

TaskFlow est une application web moderne conçue pour améliorer la productivité des freelances et des petites équipes. Avec une interface utilisateur élégante en design sombre, elle offre des outils puissants pour la gestion de projets, la collaboration en temps réel et l'optimisation des workflows grâce à l'intelligence artificielle.

## Fonctionnalités

- **Gestion de projets** : Organisez vos tâches et projets de manière intuitive
- **Collaboration en temps réel** : Travaillez ensemble avec votre équipe
- **Algorithmes intelligents** : Optimisez vos journées avec des suggestions basées sur l'IA
- **Interface sombre** : Design moderne et agréable pour les yeux
- **Responsive** : Fonctionne sur tous les appareils

## Installation

1. Clonez ce repository :
   ```bash
   git clone https://github.com/joekal/taskflow.git
   cd taskflow
   ```

2. Ouvrez `index.html` dans votre navigateur web préféré.

## Utilisation

Ouvrez simplement le fichier `index.html` dans un navigateur web moderne pour accéder à l'application.
